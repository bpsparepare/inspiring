## Instalasi

1. Clone repository ini
```bash
git clone https://github.com/username/inspire.git
```

2. Masuk ke direktori proyek
```bash
cd inspire
```

3. Install dependencies
```bash
flutter pub get
```

4. Jalankan aplikasi
```bash
flutter run
```
